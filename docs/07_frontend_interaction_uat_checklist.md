# 前端交互验收清单（UAT）

- 文档版本：1.0
- 修订日期：2026-03-20
- 适用范围：`/` 前端页面（`app/web/index.html`）

## 1. 验收准备

1. 启动服务：`./scripts/start_local.sh` 或 `uvicorn app.main:app --reload`。
2. 浏览器：Chrome 最新稳定版。
3. 数据准备：
- 至少 1 个 `SUPER_ADMIN`
- 至少 1 个 `ADMIN`
- 至少 1 个 `USER`
- 至少 1 批导入任务（包含 `RUNNING/SUCCESS/FAILED` 中至少两种状态）
- 至少 20 条审计日志

## 2. 通用交互

| 用例ID | 检查项 | 操作步骤 | 预期结果 |
|---|---|---|---|
| FE-GEN-001 | 登录状态展示 | 登录任意账号 | 右上角显示 `username / role` |
| FE-GEN-002 | 权限 Tab 可见性 | 分别登录 `SUPER_ADMIN/ADMIN/USER` | Tab 按角色正确启用/禁用 |
| FE-GEN-003 | Toast 反馈 | 执行成功与失败操作各一次 | 均出现对应成功/失败提示 |
| FE-GEN-004 | 页面切换动画 | 在各 Tab 间切换 | 面板切换流畅，无明显闪烁 |
