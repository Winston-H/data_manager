## 3. 关键路径

1. `AUTH-001` → `AUTH-002` → `UM-001`
2. `ENC-001` → `IDX-001` → `IDX-002` → `QRY-001` → `QRY-004`
3. `IMP-002` → `IMP-003` → `AUD-001`
4. `UM-002` 与 `QRY-001` 汇合后完成配额拦截闭环

## 4. 完成定义（项目级）

1. 所有 P0 任务完成并通过集成测试。
2. 安全条款（加密、权限、审计、密钥管理）全部验证通过。
3. 性能压测达到目标，或风险项有明确修复计划与时间。
4. 交付部署包、运维手册、UAT 签署记录。
